# **App Name**: MediWeb Hub

## Core Features:

- Public Landing Page: Display hospital information, doctor directory, and navigation links.
- Secure Authentication: Provide separate login/signup flows for patients and doctors.
- Patient Dashboard: Enable patients to book appointments, view test results, and access medical records.
- Doctor Dashboard: Display daily appointments, patient history, and a digital prescription tool.
- Dynamic Routing: Implement clean URL structures and seamless navigation between dashboards.

## Style Guidelines:

- Primary color: Calm sky blue (#70cbe3), symbolizing trust and health.
- Background color: Light pale blue (#e0f2f7), provides a clean and professional backdrop.
- Accent color: Soft lavender (#b289ca), creates a calming but distinct element.
- Body and headline font: 'PT Sans', a humanist sans-serif, providing a modern and slightly warm feel, suitable for both headlines and body text.
- Use clear, recognizable medical icons for navigation and key features.
- Ensure a clean, responsive layout with intuitive navigation.
- Subtle transitions and loading animations to enhance user experience.
